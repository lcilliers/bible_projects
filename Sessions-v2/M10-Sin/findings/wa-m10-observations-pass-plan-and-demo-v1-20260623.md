## 1. Demonstrator — teasing out the SPEECH object (#9 + #28)

Reading #9 (blasfēmia/blasfēmos noun) + #28 (blasfēmeō verb), the observations that emerge:

- **It is not a faculty in operation — it is an *outflow*.** Speech-sin has **faculty empty** on the lemma; its inner-being signal is carried *only* by the co-seated **heart** ("out of the heart come… slander", Mat 15:19, *kardia*). The unit is the **mouth's output**, not an inner faculty.
- **It describes the inner being *diagnostically* — the mouth reveals the heart.** The object's whole significance is that it **externalises and exposes** an inner state ("out of the abundance of the heart the mouth speaks"; the blasphemer acts "in unbelief", 1 Ti 1:13). So this object *describes* the inner being by being its **symptom / window**.
- **It carries a *direction* (an outward vector to an object):** God-ward (blasphemy against God/the Spirit) vs person-ward (slander). The inner corruption is one; the *target* differs.
- **It links act → identity:** blasfēmos "the blasphemer" ties the speech-act back to a person-type (→ the identity objects, #16/#23).
- **It has a limit-case status:** blasphemy-against-the-Spirit = the one "unforgivable" — a *status* attached to a speech-act (→ the forgiveness/record objects, #14/#13).

**Candidate label for this object-kind:** **EXPRESSION / OUTFLOW** — *the inner being's state externalised through an organ (here the mouth), which describes the inner being by exposing it.* It is the **output** of a characteristic, not the characteristic. (Political revolt #8 is the same kind via a different organ — action/relation rather than mouth.)

> This is exactly why a "sinful speech" unit *looks* like a characteristic (it is sinful, it is inner-rooted) but is not one: it is the **outflow** of a characteristic that lives elsewhere (the heart's evil — #24/#27). The unit's value is **diagnostic**, not constitutive.

## 2. The emergent picture (HYPOTHESIS — to be tested by the full pass, NOT imputed)

From the recognition read + this demonstrator, M10's objects appear to be **several distinct kinds**, each a different *way the inner being is described*:

| Candidate object-kind (label held open) | What it is | Units (provisional) |
|---|---|---|
| **CHARACTERISTIC** | a faculty/disposition *in operation* | #20, #21, #27 (+ borderline #19, #22) |
| **CONDITION / STATE** | a standing state of the inner being (not an act) | #11, #12, #24, #29, #30, #31 |
| **RECORD / LIABILITY** | the forensic weight borne before God | #13, #17, #18 |
| **IDENTITY / PERSON-TYPE** | what the person *is* at the moral core | #16, #23 |
| **EXPRESSION / OUTFLOW** | the inner state externalised (mouth, action) | #8, #9, #28 |
| **MECHANISM / ROUTE** | how sin enters/operates | #10 |
| **REMEDY / ATONEMENT** | the dealing-with mechanism | #14 (+ the atonement seam, defilement-cleansing) |
| **EXTERNAL AGENCY** | imposed from without (not the person's own interior) | the "evil one" (#24); defilement-by-spirit (#32) |

**The thesis to test:** the inner being is described not only by its *characteristics* but by the **conditions** it falls into, the **records/liabilities** it incurs, the **identities** it hardens into, the **outflows** it produces, the **mechanisms** that exploit it, the **remedies** applied to it, and the **agencies** that invade it. M10 is mostly these *other* kinds because **sin is described relationally and operationally**, not as a single faculty. Labels above are placeholders — the pass should let them sharpen or regroup.

## 3. Proposed observations pass (introduce the 17 files to AI, by group)

Run a structured observation pass, **group by candidate object-kind** (above), per group asking the same questions so observations are comparable:

1. **What kind of object is this** — characteristic, condition, record, identity, outflow, mechanism, remedy, agency, or something we have no word for yet?
2. **How does it describe the inner being** — constitutively (it *is* the inner state), diagnostically (it *reveals* it), consequentially (it *results from* it), or relationally (it *acts on/from* it)?
3. **What would we call it** — a candidate label, and what distinguishes it from the neighbouring kinds.
4. **Emergent patterns / surprises** — let the evidence push back on the hypothesis.

Then a **cross-cutting synthesis**: consolidate the labels into the §10 object-kind set (characteristic vs status vs … vs the new kinds), tested against every unit.

**Sequence:** start with the kinds that are *most clearly not characteristics* (IDENTITY #16/#23 · EXPRESSION #8/#9/#28 · MECHANISM #10) to sharpen the contrast, then the harder CONDITION / RECORD / REMEDY / AGENCY groups, then the CHARACTERISTIC trio (#20/#21/#27) last as the control.
