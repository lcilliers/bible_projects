## 4. How we build it (the method, not yet executed)

1. **Build the core (next):** read the core lemmas' verses — what does Scripture say sin *is*? Start with the Hebrew triad (ḥāṭāʾ / ʿāwōn / pešaʿ), then the NT hamart-. The core is large (~1,072 occ), so it is read in staged passes, written to md as each strand completes.
2. **Fan out:** for each ring term, read it *against the core* — use the extract's `compound` co-term web to see which core terms it co-occurs with and how it qualifies/extends/results-from/remedies the core. A ring term is never characterised alone.
3. **Then, and only then,** ask what higher constructs (status forms, and any genuine inner-being operations) the whole shows — and how to type them.
