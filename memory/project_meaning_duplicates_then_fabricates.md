**The diagnosis (le Roux, 2026-06-11).** The meaning process largely DUPLICATES the 14-dimension work; some
observations live ONLY in the meaning; re-reading the whole (partly incomplete) material each time forces
"making up some stuff because it's needed" or re-doing because incomplete — a vicious circle he has been
in for 4 months.

**Confirmed by evidence (not asserted):**
- The pipeline's system prompt ORDERS the meaning to reflect every non-null field; `SELFAUDIT` +
  `audit_paragraph()` check field→paragraph ONLY (no paragraph→field). Duplication is enforced; extra prose
  is unaudited.
- Generation split (8,367 records): 14 dimensions ~397 chars (40%), meaning ~590 chars (**60%**) — the
  paragraph costs more than all dimensions combined.
- Three real M02 records show three patterns: Mat 2:16 thumoō = PURE DUPLICATION (restates faculty tags
  verbatim); Mar 3:5 orgē = OBSERVATION-ONLY-IN-MEANING (anger-grief fusion + co-terms sullupeō/pōrōsis, in
  no dimension); Rom 1:18 orgē = IMPORTED FILLER ("gospel backdrop" pulled from 1:16-17, not the verse).

**Why it is the circle:** free prose abhors a vacuum — where dimensions are incomplete it invents; where it
notices something with no slot it adds prose. Each re-read notices/imports DIFFERENT things → meaning differs
every time → reads inconsistent/incomplete → re-do. Also FALSIFIES the prior premise that "value is safely
mirrored as ~14 queryable findings" — value is in fact prose-locked in the meaning
([[project_verse_layer_rollout_before_distill]] assumed lossless mirror; it is not).

**The fix — reverse the primacy:**
1. Dimensions PRIMARY: disciplined, option-list, silence-valid, extracted column-wise with ONE stable rubric
   against COMPLETE inputs → bounded+grounded ⇒ re-read returns the same answer ⇒ variation stops.
2. Meaning DERIVED: deterministic roll-up of the fixed dimensions; can only restate ⇒ cannot invent; import
   vector closes (cross-verse synthesis → SYNTH layer, not a verse meaning,
   [[feedback_verse_meaning_grounded_not_imported]]).
3. Promote genuine meaning-only observations to NEW dimensions: a paired/fused-with field, key co-terms, and
   the cause→effect dynamics ([[project_verse_extraction_cause_side_gap]]).

This is WHY [[project_column_wise_ve_hypothesis]] matters beyond speed/consistency: column-wise SEPARATES
disciplined extraction (must be stable) from generative prose (where invention enters) — it is the mechanism
that breaks the circle. Working note: research/investigations/wa-column-wise-ve-hypothesis-v1-20260611.md.
