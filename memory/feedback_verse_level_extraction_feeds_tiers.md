**Researcher review of `Workflow/Tiers/wa-tier-questions-extract-v1-20260604.md` against verse-level analysis
(2026-06-09).** Core finding: **the catalogue tiers are CHARACTERISTIC-level synthesis targets** (every
question says "this characteristic") — they don't suit verse-level analysis — **but the verse level feeds the
tiers.** So we need a **verse-level extraction record**, distinct from the catalogue: structured fields per
**typed-term-in-verse**, with **explicit option-lists**, that **roll up** into the tiers. The current L2
extracts only 4–5; the full set is ~13. Spec:
`research/investigations/wa-verse-level-extraction-spec-v1-20260609.md`.

**The 13 verse fields** (feeds-tier · mechanical/read): sense_applied (T7.1/T1.1.2·M) · type action/status/
quality (T1.2.1·M) · compound (T1.2.2·R) · mode/stem (T1.4.2·M) · **constitutional_location** multi-select
spirit/soul/heart/mind/will/conscience/body/NONE (T2.x·M+R) · **origin** explicit list (T2.9·R) · **faculty**
per-term multi-select from 11 (T3·R) · attributed_to_God (T0.1.2·R) · purpose_equips (T0.2.1·R) ·
typology_direction (T0.4.2·R) · immediate_response (T1.5·R) · produces_effect (T1.6→T5/T6·R) ·
directional_relational_implication (T1.1.3·R).

**Three structural fixes:**
1. **Origin = explicit checklist, ADD `from-other-spirits`** — the catalogue's origin (within / outside / God
   / generational) **excludes spiritual-being origin; it must be included** (angelic/adversarial influence is
   a real constitutional source).
2. **Faculty = restructure T3** — collapse 11 faculties × 3 Qs (33) into ONE verse field: which faculty(ies)
   the term engages, multi-select, **per-term from lexical meaning** (fixes the per-cluster induction,
   [[feedback_faculty_must_be_per_term_not_per_cluster]]); enable/impair + what-reveals are roll-up/read.
3. **Location = multi-select across ALL constitutional levels**, not just body.

**Implications:** the L2 **read's job is to produce the extraction record** (not just resolve lexical shade);
option-list fields are closed vocabularies (auditable; `NONE`/`not-stated`/`SILENT` first-class —
state-not-induce); **tiers (T0–T3) are COMPUTED roll-ups, not asked**; T4/T5/T6 are the relationship/dynamic
layer fed by produces/response + the co-occurrence array. Reframes the L2 output from the ad-hoc 4-tier write
to a 13-field record. Builds on [[feedback_characteristic_is_typed_term_in_verse]] and the catalogue coverage
analysis. **Decisions D1–D5 pending researcher confirm before rebuilding the L2 extraction.**
