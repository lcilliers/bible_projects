**From running M15 (Wisdom) through the L2 writer (2026-06-09), proving generalisation beyond M01 (Fear).**

**What generalised (the win):** the writer's structural tiers — **T7.1 lexical · T1.2 kind/type · T1.4 mode
(stem) · T2 body-locus · the triage/escalation** — all worked for a non-Fear cluster. M15: 1734 term-verses,
**93% ACCEPT / 7% ESCALATE** (cleaner than Fear — wisdom terms are less shade-ambiguous), 325 OPEN Session B
findings surfaced. The mechanical pass is cluster-agnostic.

**What broke (the finding):** the **per-cluster FACULTY** assignment **induces**. `boulomai` ("to will, wish,
choose, determine") is in the Wisdom cluster but is **volition**, not cognition — yet it got tagged
`cognition` because the cluster's primary faculty is cognition. **A cluster's primary faculty is wrong for its
off-faculty terms** (M15 mixes wisdom / understanding / counsel with intent / will terms). This violates
[[feedback_clarifications_reusable_but_state_not_induce]] — the faculty was *induced from the cluster*, not
*read from the term*.

**The rule:** **faculty (T3) must be DERIVED PER TERM from the lexical meaning** (the catalogue coverage
already said T3 is read from lexical meaning), **never assigned per cluster.** A wisdom-term → cognition; an
intent-term → volition; a conscience-term → conscience — per the term's own meaning.

**Action taken:** removed the **1734 induced M15 cognition findings** (the faculty tier is un-asserted for
M15 until per-term derivation is built — state-not-induce); kept M15's 4 structural tiers (6936 findings).
**M01 affect kept** — verified uniform (all fear/terror/dread/anxiety terms genuinely engage affect), so its
per-cluster assignment happens to be correct; but for principled consistency it should also become
per-term-derived in v2. **Next refinement: per-term faculty derivation from lexical meaning** (a
meaning→faculty map across T3.1 perception · T3.2 cognition · T3.4 affect · T3.6 volition · T3.7 agency ·
T3.8 moral-eval · T3.9 conscience · …).
