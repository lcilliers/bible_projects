**The L1-sweep read-only roll-up (2026-06-08)** correlated 5 angles across all 46 clusters. Synthesis:
`research/investigations/wa-cross-cluster-rollup-synthesis-v1-20260608.md` (+ backing: cooccurrence,
cluster-profiles, shared-forms, keyword-overlap, link-correlation docs; scripts `_assess_*` of the same
names).

**Two axes → three link classes:** co-occurrence (angle 1 = *appear together*) × keyword-Jaccard (angle 5 =
*mean the same*):
- **SAME-ish** (high both) → merge candidate / shared characteristic. e.g. Wisdom↔Strength, Sin↔Wickedness,
  Wisdom↔Righteousness, Grief↔Weakness, Love↔Strength.
- **RELATIONAL** (high co-occur, low semantic) → distinct faculties **operating together**; map the dynamics,
  don't merge. e.g. **Life↔Relational (211, 0.03)** — the strongest link, near-zero semantic; Sin↔Repentance↔
  Salvation↔Righteousness (a narrative sequence); Fear↔Strength; Joy↔Praise. **This is the bulk of the fabric.**
- **KIN** (high semantic, low co-occur) → parallel facets / boundary work. e.g. Repentance↔Transformation
  (0.23, only 22), Hope↔Trust, Truth↔Faith, Speech↔Prophecy, Pride↔Folly.

**Decisive finding:** most strong links are **RELATIONAL, not SAME** — the cross-cluster fabric is faculties
*co-operating in verses*, not clusters that should merge. **So the synergy phase is THREE operations keyed by
class (consolidate SAME-ish · map RELATIONAL dynamics · resolve KIN boundaries), not one** — the reshaping
[[project_all_clusters_l1_sweep_first]] anticipated. Of 17266 characteristic verses, 44% are cross-cluster;
every cluster touches 40–45 of the other 45. Strength is a connective *backbone* (co-occurs with all at low
%-of-self), not a tight partner.

**Other observations:** shared forms are rare but sharp (37 forms span ≥2 clusters — `ha.lal` Pride/Folly/
Praise, `a.man` Truth/Trust); the **qualifier layer (angle 2) is contaminated by function-word T2 terms**
(`al, am, a.ni, lo, gam, hu`) — genuine qualifiers (`ophthalmos`, `splanchnon`, oil) are a buried minority →
the **T2 set needs a particle/pronoun strip** before the qualifier map is trustworthy.

**Caveat:** keyword-Jaccard is absolutely low (0.02–0.23) because the keyword source isn't normalised (full
medium-def prose inflates sets) — the *ranking* is meaningful, the SAME/RELATIONAL threshold is directional;
the P1 source-clean would sharpen it. **Next (read-only): qualifier particle-strip + a synergy-group proposal
by link class, for review before any synthesis write.** Implements
[[feedback_l1_l2_is_multi_angle_report_then_synthesise]].
