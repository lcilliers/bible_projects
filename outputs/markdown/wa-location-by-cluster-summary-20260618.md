# Lexical field `location` — summary by cluster

**File:** wa-location-by-cluster-summary-20260618.md · **Type:** read-only DB summary (markdown)
**Date:** 2026-06-18 · **Source:** `ve_lexical` where `ve_label='location'` (active rows), joined to each term-in-verse's `mti_terms.cluster_code`.

`location` = the constitutional **seat** the inner-being state is seated in (spirit/breath sense-gated; read-disambiguated). Absent in a verse = not located. Multi-value possible (one row per distinct seat-level per term-in-verse), so row counts exceed verse counts.

## Corpus totals

| location seat | rows | % of located |
|---|---:|---:|
| heart | 2,604 | 45.6% |
| soul | 2,229 | 39.0% |
| spirit | 28 | 0.5% |
| flesh | 627 | 11.0% |
| UNRESOLVED | 226 | 4.0% |
| **all located** | **5,714** | 100% |

> Only **heart / soul / spirit / flesh** surface as locations across the entire corpus — `mind` and `conscience` (both in the seat-map) **never resolve** as a `location`. `spirit` is strikingly rare (28). `UNRESOLVED` = the mechanical pass expected a seat but the read could not decide which.
