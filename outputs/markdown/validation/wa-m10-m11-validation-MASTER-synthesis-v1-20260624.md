## 1. The verdict in one line

The **base lexical sense-readings are sound** — the imposition is **not** in the lexical and **not** in "what each verse says about sin." It is concentrated, ~70–85%, in the **analytical superstructure laid on top**: the §10 object-kind grid, the 32-unit structure, the cross-cluster pole-pairs, the resolution sequence, and every faculty-seatedness claim. The validators converged independently:

| Validator | Imported-meaning rate |
|---|---|
| M11 (5 units + synthesis) | 8/12 ≈ **67%** |
| M10 characteristic + condition | 5/6 ≈ **83%** |
| M10 record + identity | 5–6/8 ≈ **62–75%** |
| M10 expression/mechanism/remedy/agency | 12/14 ≈ **86%** |
| M10 structure + unit-distinctions | 8/10 ≈ **80%** |

This matches the pilot (6/7) and is the *expected* magnitude given the foundational finding.

## 2. The mechanical root cause (why it happened, provably)

`faculty`, `type`, and `lemma_meaning` are **lemma-constants** — they never vary across a term's occurrences (faculty varies in **0/86** M10 terms and **0/15** M11 terms; type in 0% of M11). The §10 object-kind grid is keyed to exactly these: "type=action → characteristic/act", "type=status → state", "faculty=X → X-seated". **So the grid relabels a fixed lemma tag as if it were a per-verse discovery from reading.** Every "this verse is conscience-seated / this unit is a characteristic" claim built on faculty/type is, mechanically, reading verse-meaning out of a constant. The legitimate per-verse evidence (sense/object/object_type/experiencer, partly valence) was bypassed.

This is the AI reflex made physical: a concept (the object-kind ontology) was adopted, then the lemma tags were harvested to populate it.

## 3. What SURVIVES — the earned, per-verse-grounded findings

Not everything fell. Where a claim rests on a field that genuinely varies per occurrence, it stands:

- **Defilement #29–32 = a non-moral CONDITION/STATE — SUPPORTED.** The one object-kind earned per-verse: `valence` varies within-lemma (neutral / forbidden / sinful), proving a durative state, not a sin. (The valence-symmetric / ritual-vs-moral split is real.)
- **M11 #2 forgiveness = act + state — SUPPORTED** (type genuinely varies: action 45 / status 3 / quality 1).
- **M11 #3 release = multi-register — SUPPORTED** (valence varies incl. sinful/forbidden; afesis type=status 16/16).
- **M11 #4 human/divine experiencer split — SUPPORTED** (experiencer varies per verse) — though "heart-seated" is TERM-CONSTANT.
- **Pole-pair sin ↔ righteousness — SUPPORTED** (81 shared verses, real co-occurrence). Defilement↔purity PARTIAL (same-verse clean/unclean link, but only ~7% — the Levitical-law verses).
- **#17 guilt reads per-verse as a conscience-OPERATION** (lev co-seated, confess-partnered) — i.e. it is **mis-filed**: a characteristic, not a "record".
- **Co-seatedness** (which other terms share the verse — nephesh/lev/conscience) is legitimate per-verse evidence (distinct from the faculty tag).

The dividing line is purely mechanical: **every SUPPORTED claim rests on a per-verse-varying field; every imposed one on a constant tag or a co-term promoted to fill a category.**

## 4. What FELL — the imposed superstructure

- **The §10 object-kind grid is largely an overlay.** CHARACTERISTIC vs CONDITION is a `type`-tag relabel (no per-verse field distinguishes them). RECORD/LEDGER is **absent from the lexical entirely** ("record/ledger/tally/inscrib" = 0 occurrences in 1,904 verses; the accounting predicate always sits on a partner verb). EXPRESSION, MECHANISM, REMEDY, EXTERNAL-AGENCY are overlays the focus-term fields don't carry — and the observation docs' own §G caveats already said so ("does NOT belong to M10's object-set", "may not be an M10 object"); the imposition was **carrying the unit into the grid despite the caveat.**
- **The 32-unit structure is largely interpretive.** Of the 7 ḥāṭāʾ units (#1–7), **only #2 Unintentional is lexically visible** (`object="unintentionally"`); the other six (wilful/habitual/suppressed/refusing/contagious) rest on verse_text or verb morphology — the `how` field that could encode manner has **0.4% coverage**. The unit files concede this ("distinguished only by reading… deferred to §10").
- **Pole-pairs written↔blot (#13) and corruption↔incorruptibility — IMPOSED** (no same-verse lexical link; constructed cross-verse synthesis).
- **All faculty-seatedness claims — TERM-CONSTANT.**

## 5. Specific corrections surfaced (factual, beyond the bias)

- **M11 resolution sequence rests on the WRONG shuv.** M11's focus shuv is H7725N = *recall-to-mind* (lay/recall heart), **not** the moral turn-from-sin verb (that is M45's). The hinge verses (Jer 18:8, Jon 3:10) contain only na.cham (obj=disaster), no turn-term. The "human turn → divine relent → atonement → reconciliation" sequence is built on a mis-identified lemma.
- **na.cham "God repents too" — CONTRADICTED.** Per-verse `object` = disaster/calamity in 23/34, never sin. God relents about *punishment*, not from sin.
- **#5 reconciliation "9th kind / state" — OVERREACH.** katallassō is action 8/8; the "state" lives only in co-term katallagē, which the programme files under **M05**, not M11.
- **a.von "tri-fold" is a sub-entry split, not one lemma read three ways** — H5771G iniquity / H5771H guilt / H5771I punishment, each near-mono-sense; lemma_meaning + type uniform across all 207 occ → a **glossing decision**, not a per-verse phenomenon. The punishment sub-entry (H5771I) is **external-pole** (penalty borne) — out of inner-being scope.
- **cha.val mis-gloss** (flagged in #21); **2Co 11:3 missing from the corpus** (#21 must-capture).

## 6. The LEXICAL defect (the researcher asked to be told)

Root-verse recourse happened repeatedly, and it always traced to **one structural lexical gap**: when a verse's key predicate lives on a **partner verb**, it is never folded into the sin-term's `how`/`object` fields, which are left empty. Examples:
- Jer 17:1 "engraved" → on cha.rash; chat.tat `how`=None.
- Rom 4:8 "count" → on logizomai; hamartia carries nothing.
- Isa 40:2 / Num 5:15 "double / remember" → on za.khar.
- The 7 ḥāṭāʾ manner-distinctions → in verb morphology (Hiphil "made to sin"), not in any field.

So the lexical is **accurate but incomplete on relational/operational predication**: `how` coverage is ~56% (M10) and **3% (M11)**; `object` ~44–74%. **Per-verse operational claims ("how sin works in this verse") have little lexical support** — which is exactly where the analysis reached for the verse text and then over-read. This is a lexical-completeness item, not a correctness error in the base senses.

## 7. Implications & recommended path (remediation DEFERRED per mandate)

1. **Demote or re-derive the §10 object-kind grid.** Either label object-kinds honestly as **lemma classifications** (type/faculty are real linguistic facts about the lemma) OR re-derive object-kinds strictly from the per-verse fields (sense/object/object_type/experiencer/valence + co-terms) — never from type/faculty. Defilement shows a kind *can* be earned this way.
2. **Treat the 32-unit structure as provisional interpretation**, not lexically-grounded fact; any §10 synthesis must re-derive unit assignments from per-verse fields, not resolve the deferred ⚖ markers from lemma-constants.
3. **Drop the per-verse faculty-seatedness claims** programme-wide (mechanically invalid).
4. **Fix the specific errors** (§5): the M11 shuv mis-identification, na.cham, #5, the a.von sub-entry framing, #17 re-file, cha.val, the 2Co 11:3 corpus gap.
5. **Lexical-completeness pass (optional):** fold partner-verb predicates into `how`/`object` if per-verse operational analysis is wanted; otherwise stop making operational per-verse claims.
6. **Carry this discipline into M12 and every cluster:** object-kind/characteristic calls must cite a per-verse-varying field, never type/faculty.
