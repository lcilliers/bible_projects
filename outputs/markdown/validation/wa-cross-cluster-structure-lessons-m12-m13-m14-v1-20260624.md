## ¬ß2. The TYPE INVENTORY emerging (the lens's types, induced from 3 clusters)
M12 alone suggested characteristic / state / expression / qualifier. M13+M14 add more, so the lens needs **at least six**:
1. **CHARACTERISTIC** ‚Äî a faculty-in-operation (M12 integrity-walk; M13 uprightness/integrity; M14 scheming).
2. **STATE / CONDITION** ‚Äî what the inner being *is* (M12 cleanness/purity; M13 truth; M14 falsehood).
3. **EXPRESSION** ‚Äî an act manifesting a characteristic, studied via its binding (M12 giving/cleansing; M13 believing/testifying; M14 slander/concealing).
4. **QUALIFIER** ‚Äî a modifier of other virtues (M12 genuineness; M13 sincerity).
5. **IDENTITY** ‚Äî a person-classification (M14 false-prophet/traitor; cf. M10 the-wicked) ‚Äî **recurs across clusters**, so it is a real type.
6. **BIVALENT FACULTY** ‚Äî a morally-neutral inner faculty directed good *or* evil **by per-verse valence** (M14 scheming/devising, cunning; cf. M10 a.von). The **valence field carries the morality**, not the lemma ‚Äî a clean per-verse-varying signal. *(New, important: some inner faculties are not intrinsically a virtue or vice; the verse's valence decides.)*
