## Preamble — What This Catalogue Is and How to Use It

### 1. Governing status

This catalogue is the master analytical instrument for the observation-question analysis phase of Session B. It governs every word study processed through the Framework B Soul Word Analysis Programme. It is read and applied at every Session B analytical run, for every word, without exception.

The catalogue does not replace the Session B instruction. It works alongside it. The Session B instruction governs the analytical process and its stages. This catalogue governs the questions asked within that process — the directions of inquiry that every word must be taken through.

### 2. An instrument of inquiry, not a checklist

Each question opens a direction of thinking. The analyst is expected to follow the direction — not merely answer the question and move on. Where answers combine to suggest something that neither produces alone — a tension, a pattern, a gap, an unexpected connection — the analyst should pause and ask the follow-on question, even if it is not in the catalogue.

New questions are welcome at any point. They arise from the data, not from the catalogue. The catalogue's job is to ensure that no direction of inquiry is systematically overlooked. The analyst's job is to go further than the catalogue wherever the data invites it.

The goal is not coverage. The goal is discovery.

### 3. The three-pass method

The catalogue is applied using three passes:

**Pass 1 — Standard application:** Apply all catalogue questions to the word's data. For each question record: ANSWER / PARTIAL / SILENT / N/A. Where the study is SILENT on a question that should have an answer, flag it [SHOULD ANSWER] with a brief note on why. This pass is systematic and complete — every question asked of every word.

**Pass 2 — Gap inspection:** Re-read the study in full after the standard pass. Identify observations in the data not reached by any existing catalogue question. For each such observation, formulate a new question using the four rules below. Add each new question to the catalogue immediately — it is now a standard question for all subsequent words. Record the observation-question pair in the current word's record. Update the master catalogue version.

**Pass 3 — Interpretive deepening:** Re-read all answers from Passes 1 and 2 together. Ask what the answers reveal in combination. Identify synergies, tensions, patterns, and follow-on questions that arise from holding the answers together. These are word-specific interpretive questions — they are recorded in the current word's analytical record but are NOT added to the universal catalogue. Pass 3 is where the depth specific to each word is developed. It does not produce new catalogue questions; it produces observations that feed Session B findings and Session C word studies.

### 4. Question discipline — four rules

All questions in the catalogue, and all new questions added to it, must satisfy these four rules:

1. **No composite questions** — one concept, one answer per question
2. **Word-neutral formulation** — the question does not name the word under study or assume its properties
3. **Verse questions as analytic probes** — not summaries of what the verse says, but questions that target a specific analytical component
4. **Etymology questions tease out the observation** — the question cannot be the observation itself

### 5. Three-axis template

Every question about what the word produces or reveals is asked across all three axes — **inner being / God / others** — regardless of whether the word has anything to say about all three. Absence of an answer on any axis is itself data. The question is always asked; the answer may be silence.

### 6. Dual function — per-word and programme-level

**Per-word function:** For each word under analysis, the catalogue is the prompt engine that determines what questions are asked. It ensures systematic coverage of all known directions of inquiry while remaining open to discovery of new ones.

**Programme-level function:** Every question asked of a word, and every result recorded, is a data point in the question-answer table in the database. Across the full corpus, these data points reveal the shape of the programme's collective understanding — which questions consistently produce answers (well-lit windows), which consistently produce silence (genuine absences or understudied areas), and which questions had not yet been discovered when early words were analysed (hidden doors). The question-answer table, queried across the corpus, is the programme's analytical memory.

### 7. Role within Session B — pointer flag resolution

Pointer flags arrive at Session B as signals generated during data preparation — something in the data triggered a question that needs to be resolved. They are not analytical findings; they are open items.

Session B's role with respect to pointer flags is:

- **Inspect each flag** against the catalogue questions. Does this flag correspond to a question the catalogue would ask?
- **If yes:** update the flag with the catalogue question reference and record the answer found in the analysis.
- **If no — error, not applicable, unsubstantiatable:** mark the flag deleted or obsolete.

By Session B close, all pointer flags for the word are resolved — either absorbed into catalogue questions with answers, or closed. The question-answer table is populated word by word as the natural output of each Session B run.

### 8. Catalogue growth and version control

The catalogue grows through Pass 2 only — genuinely new directions not yet asked of any word. Questions added during one word's Pass 2 become standard questions for all subsequent words. Each addition is recorded with the word that generated it and the date.

Every time new questions are added, the catalogue is version-incremented as a minor update. The version history records which word and session generated each increment. When new additions from successive words become rare and marginal, the catalogue is approaching completeness for the programme's current word set.

The catalogue is a living governing document. It is never frozen. Its authority comes from its completeness across the programme's accumulated analytical experience.
